Какие вы знаете scope у Spring бин?
|
1) Singleton - используется по умолчанию.
Создает только один экземпляр бина, публичные свойства класса не будут потокобезопасными;
2) Prototype - создает новый экземпляр при каждом запросе;
3) Request - аналогичен Prototype, но название служит пояснением к использованию бина в веб приложении.
Создается новый экземпляр при каждом HTTP запросе;
4) Session - создается новый экземпляр при каждой новой HTTP сессии.
