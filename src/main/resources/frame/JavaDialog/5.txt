Устройство памяти в Java.
|
1. Stack - по потокам, хранит примитивы и ссылки на объект.
2. Heap - хранит все созданные объекты. Heap делится на new gen и old gen.
3. Code Cache - это область используется JVM, когда включена JIT-компиляция.
4. Metaspace - это бывший permgen. Здесь хранится информация,
используемая JVM (используемые классы, методы).
